##总模块管理        
产品库与项目库分离管理，两个库版本不需要统一；产品库版本按功能升级范围定义版本；项目库每个项目只依赖产品库中的某一版本，单独一个版本库，按项目阶段(一期，二期等)定义版本库；        
        
产品库在初始阶段会存在功能缺陷或者BUG，在项目进行中对不能满足项目需求的功能（不包括个性化界面）修改加入到产品库的升级，以完善产品功能；项目库主要维护的部分主要是对项目个性化前台界面的修改，功能全部合并到产品库中；
